# Escrever no terminal: materiais complementares

Guias passo a passo para escrever a tese (ou dissertação, artigo, qualquer texto longo) **fora do Word**, usando ferramentas que normalmente ficam no mundo da programação: **Quarto, Git e Neovim**.

São o material de apoio do vídeo [*Um psicólogo escrevendo a tese no terminal*](https://www.youtube.com/watch?v=UhWpCMy9o-E). Foram escritos **para quem nunca programou**: gente de psicologia, humanas, gente que escreve prosa de verdade. Não há pré-requisito nenhum.

> A tese por trás de tudo: **a ferramenta em que você escreve não é neutra.** Ela molda a sua atenção e a sua coragem. Um editor que te deixa mexer no texto sem fricção, e sem medo de "estragar a formatação", é um editor que te deixa pensar com a cabeça mais livre.

## Os materiais

| Arquivo | O que é |
|--------|---------|
| [`quarto.md`](quarto.md) | Escrever em texto puro e gerar o PDF formatado sozinho (APA, citações e referências automáticas). Sem nunca encostar no LaTeX. |
| [`git-basico.md`](git-basico.md) | Guardar o histórico completo do texto e reescrever sem medo: a versão anterior está sempre a um comando de distância. |
| [`curso-neovim.md`](curso-neovim.md) | Um curso progressivo do iniciante ao fluente, focado em quem **escreve prosa**, não código. |
| [`nvim-cheatsheet.pdf`](nvim-cheatsheet.pdf) | Folha de referência dos comandos do Neovim, pronta pra imprimir. (Fonte: [`nvim-cheatsheet.tex`](nvim-cheatsheet.tex).) |

## Como baixar

- **O pacote inteiro:** botão verde **`Code` → `Download ZIP`** aqui em cima.
- **Um arquivo só:** clique nele na lista acima e use o botão de download (ou "Raw").
- **Com Git** (depois de ler o `git-basico.md` 😉):
  ```
  git clone https://github.com/pedro-moser/escrever-no-terminal.git
  ```
